# paperLens
A system for structured parsing and analysis of scientific papers, which automatically converts PDF papers into structured representations including sections, figures, tables, and experimental results for downstream comparison and understanding. 一个面向科研论文的结构化解析与分析系统，可将PDF论文自动转化为包含章节、图表、实验结果等信息的结构化表示，用于后续的跨论文对比与分析
