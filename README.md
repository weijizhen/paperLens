# paperLens
A system for structured parsing and analysis of scientific papers, which automatically converts PDF papers into structured representations including sections, figures, tables, and experimental results for downstream comparison and understanding. 一个面向科研论文的结构化解析与分析系统，可将PDF论文自动转化为包含章节、图表、实验结果等信息的结构化表示，用于后续的跨论文对比与分析

## 🧠 Overview / 项目简介

**EN:**  
PaperLens is a system for parsing and analyzing scientific papers in PDF format. It extracts structured information such as sections, figures, tables, and experimental results for further comparison and analysis.

**中文：**  
PaperLens 是一个面向科研论文的 PDF 解析与分析系统，用于提取论文中的章节、图表、表格和实验结果，并支持后续的结构化分析与对比。

---

## 🎯 Goal / 目标

**EN:**  
Help researchers understand and compare papers more efficiently by converting unstructured PDFs into structured data.

**中文：**  
将非结构化的 PDF 论文转化为结构化数据，帮助更高效地理解和对比不同论文。

## 🏗️ Pipeline / 系统流程
PDF Paper

↓

Structured Data Output
